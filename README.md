# FroggyMemo
[바로가기](https://memoapp.frogit.xyz)

## 프로젝트 설명
Vue.js 실습용으로 만들고 있는 메모 애플리케이션입니다.
Web Memo Application with Vue.js

## 기능

### 현재 기능
- 메모 추가, 수정, 삭제
- 고유 ID, 제목, 타임스탬프 포함
- 클라이언트측 저장 기능(IndexedDB)
- 설정 기능
- 다국어 지원

### 추가 예정 기능
- ID(생성시간)순, 수정시간순, 제목 가나다순 정렬
- 마크다운
- 리마인더
- 상단 핀
- 메모색 설정
- 배경 테마 등 모양 설정
- 휴지통
- 데이터 내보내기/가져오기
